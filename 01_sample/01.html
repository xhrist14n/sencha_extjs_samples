Iniciando Extjs Primer ejemplo de Hola Mundo

Dentro de lo que es programación con javascript hay muchos Frameworks como son jquery, scriptaculus, etc., etc. En fin son muchos y tienen funcionalidades muy distintas el uno del otro, por tanto no es difícil encontrar una gran variedad de Frameworks; para todos los fines inimaginables, ahora claro hay muchos pero pocos destacan, un claro ejemplo de esto es Sencha Extjs, el cual nos permite hacer aplicaciones Web de tipo RIA (Rich Internet Application) este tipo de aplicaciones Web hechas con Sencha Extjs tienen un peculiar aspecto que es poder funcionar como una aplicación de escritorio con ventanas modales, grillas , graficas de reportes, formularios en ventana, entre otras muchas funciones.

Instalacion de Extjs 4.2

El modo de instalacion es sencillo lo primero que debemos hacer es descargar nuestro framework lo cual lo podemos hacer desde la pagina sgte: http://www.sencha.com/products/extjs/download/, luego de haber descargado nuestro framework debemos descomprimir su contenido en una carpeta accesible desde donde podamos acceder facilmente. Es recomendable usar la version GPL, tambien hay una version comercial para fines didacticos se usara la version gpl para nuestras pruebas.

Creacion de codigo hola Mundo

Seguidamente en un directorio que este un nivel de la carpeta donde estan ubicados los archivos del framework Extjs procedemos a crear una carpeta que puede llamarse 01_sample donde crearemos un archivo html llamado sample.html donde redactaremos el siguiente codigo:

<html> <head> <link rel="stylesheet" href="../extjs/resources/css/ext-all.css" type="text/css" /> <script type="text/javascript" src="../extjs/ext-all.js"></script> <script type="text/javascript" src="sample.js"></script> </head> <body> </body> </html>

Luego redactamos en archivo js llamado sample.js desde donde se ejecutara las acciones que necesitamos para mostrar nuestro cuadro de dialogo donde se mostrara nuestro mensaje de Hola Mundo.

Ext.onReady( function(){ Ext.Msg.show( { title:'Mensaje', msg: 'Hola mundo', buttons: Ext.MessageBox.OK, fn: function(){ console.log("esto funciona"); return; }, icon: Ext.MessageBox.QUESTION } ); return; } );

Visualizacion de Ejemplo

Y por ultimo podremos ver al abrir la locacion del archivo sample.html en nuestro server de http, el dialogo de hola mundo deseado.
